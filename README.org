* Website

This is my about me page, available at pwalsh.me

** Quickstart

Dependencies:

#+BEGIN_SRC sh
lein deps
npm install
#+END_SRC

Development:

#+BEGIN_SRC sh
npx webpack --watch
lein serve
#+END_SRC

Build:

#+BEGIN_SRC sh
npx webpack
lein build
#+END_SRC

Deploy:

#+BEGIN_SRC sh
gsutil rsync -d -r output gs://pwalsh.me
#+END_SRC
